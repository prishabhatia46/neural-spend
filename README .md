## рҹ§© Key Functions

| Function | What it does |
|---|---|
| `showTab(name)` | Switches between all 7 tabs |
| `renderBars()` | Animated monthly spending bar chart |
| `renderDonut()` | Interactive SVG donut chart |
| `renderTxns()` | Live transaction list (dashboard) |
| `renderTxnFull()` | Full filterable transactions tab |
| `renderBudgets()` | Budget progress bars with overspend |
| `renderGoals()` | Circular SVG savings goal rings |
| `renderInvestments()` | Portfolio holdings list |
| `runAI()` | Cycles AI-generated insights |
| `smartSuggest()` | AI suggestion button animation |
| `openModal()` | Detail popup for any item |
| `filterTxn()` | Filter transactions by category |
